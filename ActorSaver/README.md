## How to Use

1. **Open the tool** - Simply open `index.html` in your web browser
2. **Enter your script** - Paste your monologue or script in the text area
3. **Click "Process Script"** - This prepares your text for all three methods
4. **Choose your method** - Use the navigation tabs to switch between methods

### First Letter Method
- Each word is replaced by its first letter
- Click individual letters to reveal the full word
- Use "Reveal All" to check your memory
- Use "Hide All" to start fresh

### Line Buildup Method  
- Start with the first word, then gradually add more
- Click "Start Practice" to begin
- Use "Next Level" to add more words
- Perfect for building confidence progressively

### Backwards Method
- Learn your lines backwards for complete mastery
- Type the word that comes before the displayed word
- Get immediate feedback on your answers
- If you know them backwards, you've got them down!

## Why These Methods Work

- **First Letter Method**: Gives you just enough to trigger memory without giving away the answer
- **Line Buildup Method**: Builds confidence by starting simple and adding complexity
- **Backwards Method**: Ensures complete mastery - if you know it backwards, you know it forwards
